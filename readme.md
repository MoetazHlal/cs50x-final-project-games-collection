## Description
This project is a collection of three simple games built using C# and Windows Forms.
The goal of this project is to demonstrate understanding of programming fundamentals,
event-driven programming, and basic game dev.

## Games Included
- Guess The Number game
- Tic Tac Toe game
- Rock Paper Scissors game

## Technologies Used
- C#
- .NET Windows Forms
- Visual Studio

## Features
- Score system
- Random number generation
- Enum usage for game states (Used for Tic Tac Toe game)
- Simple and clean UI

## How to Run
1. Open the solution file in Visual Studio
2. Build the project
3. Run the application
